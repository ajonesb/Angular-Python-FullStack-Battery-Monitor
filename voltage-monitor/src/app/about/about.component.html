About This Project

This project was created to monitor voltage data from batteries using Angular for the frontend and Python - FastAPI for the backend.

Challenges faced: One of the main challenges was parsing and processing the complex JSON structure from the measurements file. Another challenge was ensuring proper error handling and data flow between the frontend and backend.

We used standalone components in Angular to focus more on the business logic and reduce the constraints associated with the traditional NgModule approach. This decision allowed for a more streamlined development process, as it eliminated the need for excessive modularization that can sometimes be a challenge when using Angular's module system. Standalone components provided a more direct and intuitive way to build our application, making it easier to reason about component dependencies and imports.

Time taken to complete: Approximately 8 hours.

Shortcomings and next steps: The current implementation could be improved by adding more robust error handling, implementing unit tests, and enhancing the UI with more interactive features. Future steps could include adding real-time data updates, implementing data caching strategies, and optimizing the backend for larger datasets.
